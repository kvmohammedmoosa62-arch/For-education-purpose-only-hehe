// 02 · YOUR PATH

Choose Your Roadmap

Master the art of breaking systems legally. Offensive security practitioners — penetration testers, red teamers — think like attackers to expose weaknesses before criminals do.

  • 01
    Fundamentals of Networking & OS
    Learn TCP/IP, DNS, HTTP/S, how packets travel, subnetting, and how Linux & Windows work internally. This is your foundation — skip it and you'll crumble later.
    CompTIA Network+Linux BasicsNetworking Fundamentals
  • 02
    Programming & Scripting
    Learn Python for automation and exploit scripting. Understand Bash for Linux workflows. Basic knowledge of C helps when reading exploits or shellcode.
    PythonBashC Basics
  • 03
    Security Fundamentals
    Understand CIA Triad, common vulnerability types (OWASP Top 10, CVEs), cryptography basics, authentication mechanisms, and the attacker lifecycle (Cyber Kill Chain).
    OWASP Top 10CompTIA Security+Cryptography
  • 04
    Web Application Hacking
    Learn to find and exploit SQLi, XSS, CSRF, IDOR, SSRF, broken auth, and business logic flaws. Practice on DVWA, Juice Shop, and HackTheBox web challenges.
    Burp SuiteDVWAPortSwigger Labs
  • 05
    Network Penetration Testing
    Perform active reconnaissance, port scanning, service enumeration, exploitation of vulnerable services, pivoting, and lateral movement in lab environments.
    NmapMetasploitWireshark
  • 06
    Active Directory & Windows Attacks
    Master Kerberoasting, Pass-the-Hash, BloodHound, LDAP enumeration, privilege escalation on Windows — the bread and butter of enterprise red teaming.
    BloodHoundMimikatzPowerView
  • 07
    CTFs, Labs & Certifications
    Solve Capture The Flag challenges, grind HackTheBox and TryHackMe machines, then validate your skills with OSCP — the gold standard of offensive certifications.
    HackTheBoxTryHackMeOSCPeJPT

The defenders keep the lights on. Defensive security is about detection, response, and hardening — making the attacker's job as painful and costly as possible.

  • 01
    Networking & OS Deep Dive
    Understand network protocols at packet level, firewall rules, routing, VLANs, and both Windows and Linux system internals including processes, memory, and permissions.
    WiresharkPacket TracerLinux CLI
  • 02
    Security Monitoring & Logging
    Learn to collect, parse, and analyze logs from firewalls, endpoints, and servers. Understand Windows Event IDs and Linux syslog for detecting malicious activity.
    SplunkELK StackWindows Event Logs
  • 03
    Security Operations Center (SOC)
    Learn SOC workflows: alert triage, escalation procedures, working with SIEMs and ticketing systems. Understand Tier 1, 2, 3 analyst responsibilities.
    SIEMSOC AnalystTryHackMe SOC Path
  • 04
    Threat Intelligence
    Understand threat actor TTPs using MITRE ATT&CK framework. Learn to consume threat intel feeds, write detection rules (Sigma/Yara), and threat hunt proactively.
    MITRE ATT&CKYARASigma
  • 05
    Incident Response (IR)
    Master the IR lifecycle: preparation, detection, containment, eradication, recovery, and lessons learned. Practice tabletop exercises and live IR simulations.
    VelociraptorTheHiveNIST IR Framework
  • 06
    Digital Forensics (DFIR)
    Analyze disk images, memory dumps, network captures, and browser artifacts. Reconstruct attacker timelines and preserve evidence correctly for legal use.
    AutopsyVolatilityFTK Imager
  • 07
    Certifications & Career Path
    Validate your defensive skills with industry-recognized certifications and build a portfolio with documented IR cases and detection engineering projects.
    CompTIA CySA+BTL1GCIHGCIA

Security researchers push the frontier — finding zero-days, publishing CVEs, reverse engineering malware, and advancing the knowledge of the entire field. This path demands deep technical mastery.

  • 01
    Low-Level Programming
    Master C, C++, and Assembly. Understand memory layout (stack, heap, BSS), calling conventions, pointers, and manual memory management — the language of exploits.
    C/C++x86 AssemblyNASM
  • 02
    Reverse Engineering
    Learn to disassemble and decompile binaries. Analyze unknown executables, understand obfuscation techniques, and reconstruct source logic from machine code.
    GhidraIDA ProBinary Ninja
  • 03
    Binary Exploitation
    Study buffer overflows, format string bugs, use-after-free, heap exploitation, ret2libc, ROP chains, and how to bypass modern mitigations like ASLR, DEP, and stack canaries.
    GDB + PEDApwntoolspwn.college
  • 04
    Malware Analysis
    Set up safe sandbox environments, perform static and dynamic analysis of malware samples, extract IOCs, understand packing/unpacking, and analyze C2 communications.
    Cuckoo SandboxANY.RUNFLOSS
  • 05
    Vulnerability Discovery
    Learn fuzzing with AFL++ and libFuzzer, code auditing, static analysis tools, and how to systematically hunt for bugs in real open-source and commercial software.
    AFL++SanitizersSemgrep
  • 06
    CVE Research & Responsible Disclosure
    Understand the full vulnerability lifecycle — discovery, PoC development, coordinated disclosure, CVE assignment, and publication. Start with Bug Bounty programs as training ground.
    HackerOneBugcrowdCVE Process
  • 07
    Publish, Present & Contribute
    Write technical blog posts, present at DEF CON or local security meetups, contribute tools to open source, and build a public research portfolio that speaks for itself.
    DEF CONBlack HatGitHubOSCP
